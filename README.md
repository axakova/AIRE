# AIRE
Scripts used to generate and analyze Systematic and proactive evaluation of AIRE missense variant effects

This script generates performs all analysis and generates figures for the manuscript. If other scripts are used, input bash scripts are provided and explained in the .Rmd script. This repository contains all dependency files.

The top comment for each chunk describes which corresponding Figure is generated. Specific figures can be searched for.

To run the script: 
1. Download repository
2. Run .Rmd file in Rstudio within the same folder as all the dependent files
